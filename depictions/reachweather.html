ReachWeather

Description

  • ReachWeather puts the current weather conditions in your Reachability view (on supported devices).

  • Weather data courtesy of the OpenWeatherMap API.

  • NOTE: This is a beta release. I don't guarantee that everything will work right.

Changelog for version 1.3.1-58 (which will eventually be version 1.4):

  • • Fixed issue where the city name woud be displayed as written in Settings (which caused issues for those who used zip codes)

  • • Potential fix for weather location accuracy issues.

  • • Cleaned up resources bundle for weather location accuracy issues.

  • • Added experimental support for weather background images (like in Weather.app).

  • • Added English, German, and Spanish localizations.

  • • Added weather images instead of text descriptions for forecasts.

  • • Added support for centering the simple weather view.

  • • Added this depiction back to the control file (for beta releases).

How to use ReachWeather

  • • Navigate to Settings > ReachWeather

  • • Type the name of your city into the space provided and hit "Return".

  • • Hit the "Back" button.

  • • You can now activate Reachability (by double-tapping TouchID) and view your current weather conditions while in any app (no Springboard support yet).

Compatibility

  • ReachWeather works with iOS 8.0+ and on any device that has Reachability enabled.
