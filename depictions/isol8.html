Description

  • Isol8 supercharges the native iOS 8 group message muting functionality, blocking ALL notifications from muted group messages, including banner, notification center, and lock screen alerts.

Changelog for version 0.1

  • • Added settings for muting NC, LS, and banner notifications (yes, I know this will mark the third time these toggles have been available. They just seem to cause a lot of issues.)

  • • Fixed issue where other switches wouldn't reload immediately after enabling/disabling the killswitch

  • • Cleaned up dead code from older versions

  • • Added a better-looking settings icon

How to use Isol8

  • • Open Settings > Isol8 and enable it if it isn't already.

  • • Go into a group message in Messages.

  • • Tap "Details" in the top right.

  • • Scroll down until you see the "Do Not Disturb" option; turn this on.

  • • Isol8 will (by default) block all banners, notification center alerts, and lock screen alerts for this group.

Compatibility

  • Isol8 works with iOS 8.0+ and all iDevice models.
